```

```

```
如果没有记录请求会继续到路由器上，路由器上有自己的DNS缓存；
```

```
如果没有记录就会到ISP的DNS缓存中查看记录；
```

```
如果没有记录就会在ISP的DNS服务器从根服务器域名服务器开始递归搜索最后得到IP地址;
```
